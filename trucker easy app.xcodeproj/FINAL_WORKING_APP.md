### PASSO 1: Criar Projeto Novo
```
1. Xcode > File > New > Project
2. iOS > App
3. Nome: "Trucker Easy"
4. Interface: SwiftUI
5. Language: Swift
```

### PASSO 2: Copiar Arquivos
```
1. Delete ContentView.swift
```


### PASSO 4: Build & Run
